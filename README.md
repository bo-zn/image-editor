# Image Editor Project

## 项目简介

这是一个基于 Vue.js 的图像编辑器项目，提供了图像剪裁、滤镜应用等功能。项目使用了 `fabric.js` 库来实现图像处理。

## 功能特性

- 图像剪裁
- 图像滤镜应用（亮度、对比度、饱和度等）
- 图像上传和下载
